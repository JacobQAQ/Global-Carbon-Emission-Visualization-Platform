# 全球碳排放与气候责任交互式数据看板（1750–2024，含 2025–2035 趋势预测）

基于 Our World in Data（OWID）CO₂ 数据集制作的单页交互式课程大作业。项目使用 Python、Pandas、NumPy、Pyecharts 和 Jinja2，将地图、排名、趋势、堆叠面积图、气泡散点图、热力图、桑基图、玫瑰图与词云整合到一个静态 HTML 看板中，并支持亮色 / 暗色主题切换。

## 数据来源

- 数据集：OWID CO2 dataset
- 文件名：`owid-co2-data.csv`
- 官方仓库：https://github.com/owid/co2-data
- 本项目不会修改原始 CSV 文件。

## 运行方式

```bash
cd co2_dashboard
python -m pip install -r requirements.txt
python build_dashboard.py
```

默认会依次在当前运行目录、脚本目录和脚本上一层目录查找 `owid-co2-data.csv`。也可以显式指定路径和输出位置：

```bash
python build_dashboard.py --data ../owid-co2-data.csv --output output/co2_dashboard.html
```

预测结束年份默认为 2035，可通过参数调整：

```bash
python build_dashboard.py --forecast-end 2040
```

生成结果：

```text
output/co2_dashboard.html
```

页面通过 CDN 加载 ECharts、世界地图和词云扩展，首次打开时需要可访问互联网。

## 预测方法说明

本看板新增的预测模块采用简单、透明的回归外推：

- CO₂、来源排放、隐含碳贸易、温度变化贡献：使用最近历史窗口的线性回归。
- GDP、人口：使用对数线性回归。
- 排放、人口、GDP、温度贡献等非负指标会限制为不小于 0。
- `trade_co2` 保留正负方向，因为正值和负值分别代表净进口 / 净出口隐含碳。
- 预测值仅用于课程展示和趋势探索，不代表正式气候模型、政策情景或 IPCC 情景预测。

## 数据口径与限制

- 2024 年适合用于排放地图、排名、人均排放和历史责任等横截面展示。
- GDP 历史数据主要更新到 2022 年，因此经济规模相关图表中的后续年份属于模型预测。
- `trade_co2` 历史可用年份截至 2023 年，后续年份为趋势预测。
- `iso_code` 为空或以 `OWID_` 开头的世界、洲、收入组等汇总实体不进入国家地图和国家排名；全球趋势允许保留 `World`。
- 能源来源分解存在统计覆盖差异，空值不等于零。堆叠面积图为保持序列连续会在绘制时处理空值，桑基图则过滤空值和非正值。
- 人均排名和气泡散点图仅纳入人口不少于 500 万的经济体，以避免微型产油经济体主导视觉空间。
- 可再生能源转型分析需要额外补充 OWID Energy 数据集，本 CO₂ 数据集不足以单独支撑完整能源转型结论。
